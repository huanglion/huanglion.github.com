```bash

```

```bash
$ cd gitUsergitUser  #默认进入的目录就是主分支master
```
```bash
$ git branch         #打印出当前分支
```
```bash
$ git branch -a      #打印出所有分支
* master
  source
  remotes/origin/HEAD -> origin/master
  remotes/origin/master
  remotes/origin/source
```
